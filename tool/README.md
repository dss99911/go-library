

Install
===
- install on the source code directory
    
    ```go install```
    
- install on any path
    
    ```go install <repository-name><source-path>```
    
    example for $GO_PATH/src/github.com/user/hello.go
    
    ```go install github.com/user/hello```
    

Test
===
````go test````

```go test github.com/user/stringutil```
